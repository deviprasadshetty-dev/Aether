System Architecture

A dual-layer approach. A local Node.js MCP server provides the interface to your AI Agent (via stdio JSON-RPC), while a Chrome Extension Service Worker executes the CDP commands directly within the browser, delivering the visual overlay and real-time execution. 

┌─────────────────────┐
│   AI Agent (MCP)    │
│  (Claude, etc.)     │
└─────────┬───────────┘
          │ stdio (JSON-RPC)

│   Aether MCP Server │

Quick Start Installation

Get the Aether MCP Server and Chrome Extension up and running in minutes.

1. Build the Server

Clone the repository and build the local Node.js MCP server.

cd server
npm install
npm run build

2. Load the Extension

Install the local Chrome extension to enable visual grounding and CDP commands.

  • Open chrome://extensions
  • Enable Developer Mode
  • Click "Load Unpacked" and select the extension/ folder

3. Configure MCP Client

Add Aether to your AI client (like Claude Desktop).

"mcpServers": { "aether-browser": { "command": "node", "args": ["path/to/server/dist/index.js"] } }
